Önemli Git Terimleri

Repository Kısaca repo, dosyaların bulunduğu projeler
Master projelerimizin kararlı sürümleri, ana gövdesi
Branch git, ağaç mantığı ile çalışır.Proje birden fazla dala ayrılıp bu şekilde geliştirebilir. Sonrasında bu dallar yine master da birleştirilip proje tamamlanır.
Merge Branch'lerde değişikliklerin masterda birleştirilme işlemidir
Push bilgisayardaki kodlarımızı githubtaki repoya gördermemizi sağlar
Pull Github daki projelerimizin son halini bilgisayarımıza çekmemizi sağlar
.gitigore Gitin takip etmek istediğimiz dosyaları(localde denediğimiz rsimler ve benzeri) push işleminde repoya göndermesini engellememizi sağlar.
